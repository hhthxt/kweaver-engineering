## 4. 建议参会角色

| 角色 | 需要确认的问题 |
|---|---|
| 业务负责人 | 业务目标、范围、成功指标 |
| 一线业务人员 | 当前流程、痛点、材料和真实案例 |
| 审核 / 合规人员 | 判断标准、审批、拒绝边界 |
| 系统 / 数据负责人 | 系统、数据来源、接口、权限 |
| 项目负责人 | 范围、里程碑、资源和验收 |

## 5. 会前 / 会后材料索要清单

| 材料 | 用途 |
